# Stack do Projeto

## Objetivo

Este documento define as principais tecnologias utilizadas no desenvolvimento do sistema da AMA - Escola de Música Gospel.

---

## Frontend

### React

Biblioteca JavaScript utilizada para construir a interface do usuário.

### Vite

Ferramenta utilizada para criar e rodar o projeto React com mais rapidez.

### React Router

Biblioteca utilizada para criar rotas e páginas dentro do frontend.

### Axios

Biblioteca utilizada para fazer requisições HTTP para o backend.

---

## Backend

### Python

Linguagem principal utilizada no backend.

### FastAPI

Framework utilizado para criar a API do sistema.

### Uvicorn

Servidor utilizado para executar a aplicação FastAPI.

---

## Banco de Dados

### SQLite

Banco de dados inicial utilizado durante o desenvolvimento local.

### PostgreSQL

Banco de dados previsto para uso em produção.

---

## Autenticação

### JWT

Tecnologia utilizada para autenticação dos usuários e proteção de rotas.

---

## Controle de Versão

### Git

Ferramenta utilizada para versionamento do código.

### GitHub

Plataforma utilizada para hospedagem do repositório.

---

## Documentação

### Markdown

Formato utilizado para escrever a documentação do projeto.

---

## Deploy Futuro

### Vercel

Possível plataforma para publicação do frontend.

### Render ou Railway

Possíveis plataformas para publicação do backend e banco de dados.
