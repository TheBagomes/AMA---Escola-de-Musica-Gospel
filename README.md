# AMA Escola de Música Gospel - Sistema de Gestão

Sistema web desenvolvido para auxiliar a gestão da **AMA - Escola de Música Gospel**, permitindo o controle de alunos, professores, matrículas, aulas, presença, disponibilidade de professores e pagamentos.

## Objetivo

O objetivo do sistema é centralizar e facilitar a administração da escola, oferecendo uma solução simples, organizada e eficiente para o acompanhamento das atividades acadêmicas e financeiras.

## Funcionalidades previstas

- Cadastro de alunos
- Cadastro de professores
- Controle de matrículas
- Agenda de aulas
- Disponibilidade dos professores
- Controle de presença e faltas
- Controle de pagamentos
- Confirmação de aulas
- Login com diferentes perfis de usuário

## Perfis de usuário

- Administrador
- Professor
- Aluno ou responsável

## Tecnologias previstas

- React
- Python
- FastAPI
- SQLite
- PostgreSQL
- Git e GitHub
